# 智能文档生成工具 - 学习笔记目录

## 🚀 自动生成功能
现在可以使用自动生成脚本创建每周学习笔记，脚本会根据当前时间自动生成文件名和内容：

```bash
# 进入学习笔记目录
cd learning_notes

# 自动创建当前周笔记（推荐）
./create_weekly_note.sh

# 创建指定周数的笔记
./create_weekly_note.sh -w 1 -y 2024

# 查看帮助信息
./create_weekly_note.sh --help
```

### 自动生成特性：
- 📅 **智能文件名**：自动生成 `YYYY-MM_week_W.md` 格式的文件名
- 🗓️ **自动计算**：根据周数自动计算正确的月份和日期范围
- 📝 **内容填充**：自动填充周数、月份、日期范围等基本信息
- ⚡ **快速创建**：一键生成完整的每周学习笔记模板

## 📚 目录说明

本目录用于记录智能文档生成工具项目的学习过程和进展。通过系统性的学习笔记，跟踪学习曲线、记录问题解决方案、总结学习收获。

## 📁 文件结构

### 模板文件
1. **`weekly_goal_template.md`** - 每周学习笔记模板
   - 专注于每周目标和完成情况
   - 使用待做列表表示进度
   - 包含学习收获、代码进展、问题解决等模块

### 自动化工具
1. **`create_weekly_note.sh`** - 每周笔记自动生成脚本
   - 根据当前时间自动生成文件名和内容
   - 支持指定周数、年份、月份
   - 自动计算正确的日期范围
   - 提供文件预览和验证功能

### 示例文件
1. **`week_1_2026.md`** - 第一周学习笔记样例
   - 展示如何使用模板
   - 包含实际的学习内容和代码示例
   - 可作为参考创建自己的笔记

2. **`2026-03_week_13.md`** - 自动生成的笔记示例
   - 展示自动生成脚本的输出格式
   - 包含自动填充的基本信息
   - 文件名格式：`YYYY-MM_week_W.md`


## 📝 使用指南

### 1. 每周学习流程
```
周一：制定本周学习目标
周二～周五：每日学习并记录要点
周六：整理学习内容，编写代码
周日：完成笔记，制定下周计划
```

### 2. 笔记创建步骤（手动方式）
1. **复制模板**：每周开始时复制 `weekly_goal_template.md`
2. **命名规范**：使用 `week_[周数]_[年份].md` 格式，如 `week_2_2026.md`
3. **填写基本信息**：设置学习周期、日期范围、学习时长
4. **设定目标**：明确本周的学习和项目目标
5. **每日记录**：在学习过程中记录关键点
6. **周末总结**：整理完成情况，分析学习收获

### 3. 笔记创建步骤（自动方式）
1. **运行脚本**：`./create_weekly_note.sh` 自动生成当前周笔记
2. **查看预览**：脚本会显示生成的文件信息和内容预览
3. **编辑内容**：打开生成的笔记文件，填写具体的学习内容
4. **保存更新**：在学习过程中不断更新和补充笔记内容

### 4. 笔记内容要点

#### 必填内容
- ✅ 本周目标与完成情况
- ✅ 实际完成的任务清单
- ✅ 学习收获和关键突破
- ✅ 代码进展和问题解决
- ✅ 下周计划

#### 可选内容
- 🔄 详细的学习曲线分析
- 📊 详细的时间统计
- 🎯 个人学习方法的反思

### 5. 命名约定

#### 手动创建命名
```
week_1_2024.md      # 2024年第1周
week_2_2024.md      # 2024年第2周
week_3_2024.md      # 2024年第3周
# 以此类推...
```

#### 自动生成命名
```
2024-01_week_1.md   # 2024年1月第1周
2024-03_week_13.md  # 2024年3月第13周
2024-12_week_52.md  # 2024年12月第52周
# 格式：YYYY-MM_week_W.md
```

**注意**：自动生成的文件名包含月份信息，便于按时间排序和查找。

### 6. 版本控制
- 每周日提交当周的学习笔记
- 使用有意义的提交信息，如："docs: 添加第1周学习笔记"
- 保持笔记的持续更新和版本跟踪

### 7. 自动生成脚本选项
```
选项:
  -w, --week <数字>     指定周数（可选，默认使用当前周数）
  -y, --year <数字>     指定年份（可选，默认当前年份）
  -m, --month <数字>    指定月份（可选，默认当前月份）
  -t, --template <文件> 指定模板文件（可选）
  -o, --output <文件>   指定输出文件（可选）
  -h, --help            显示此帮助信息

示例:
  ./create_weekly_note.sh                    # 自动创建当前周笔记
  ./create_weekly_note.sh -w 1              # 创建第1周笔记
  ./create_weekly_note.sh -w 2 -y 2024      # 创建2024年第2周笔记
  ./create_weekly_note.sh -o my_note.md     # 自动创建并指定输出文件
```
